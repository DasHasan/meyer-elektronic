# MYR Meyer Elektronic Website

A responsive HTML/CSS website recreation for Meyer Elektronic GER.

## Files

- `index.html` - Main HTML structure
- `styles.css` - Complete styling with responsive design
- `script.js` - JavaScript for interactivity

## Features

### Responsive Design
- **Desktop**: Full layout with multi-column grids
- **Tablet** (≤768px): Adjusted layouts, 2-column grids
- **Mobile** (≤480px): Single column, hamburger menu

### Sections Included

1. **Header** - Sticky navigation with logo and menu
2. **Hero Section** - Large logo display
3. **Red Banner** - Call-to-action banner
4. **About Section** - Company information with image
5. **Services Icons** - Three main service highlights
6. **CTA Gray Section** - Contact call-to-action
7. **Detailed Services** - Four service categories
8. **Contact Form** - Responsive contact form
9. **Team Section** - Team member profiles
10. **Location Section** - Map and contact information
11. **Red Wave Section** - Decorative section
12. **Footer** - Detailed footer with company info

### Interactive Features

- Responsive hamburger menu for mobile
- Smooth scrolling for anchor links
- Form submission handling
- Scroll animations for elements
- Sticky header with shadow on scroll
- Hover effects on buttons and links

## Usage

Simply open `index.html` in a web browser to view the website.

```bash
# Open in default browser (Linux)
xdg-open index.html

# Or serve with a simple HTTP server
python3 -m http.server 8000
```

Then navigate to `http://localhost:8000`
